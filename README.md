# Komplexní čísla – Prezentace

-> **[Spustit prezentaci](https://davidhovorka.github.io/komplexni-cisla/)**

Tato interaktivní prezentace vizualizuje základy komplexních čísel. Je vytvořená pomocí matematického enginu [Manim](https://www.manim.community/) a [Manim Slides](https://github.com/jeertmans/manim-slides).

## Jak prezentaci ovládat
* **Šipka vpravo** - krok vpřed
* **Šipka vlevo** - krok zpět
* **F** - zapnutí režimu celé obrazovky
* **ESC** - vypnutí režimu celé obrazovky
* **G** - skočit na snímek podle pořadí
